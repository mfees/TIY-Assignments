### 1. `<ul>` -- Unordered List

* *type:* block-level
* *content:* zero or more <li> elements, eventually mixed with <ol> and <ul> elements.
* *support:* 
* *example:*
```
<ul>
  <li>Thing One</li>
  <li>Thing Two</li>
</ul>
```
<ul>
  <li>Thing One</li>
  <li>Thing Two</li>
</ul>

### 2. `<ol>` -- Ordered List

* *type:* block-level
* *content:* Zero or more <li> elements
* *support:* Chrome, Firefox, Safari
* *example:*
```
<ol>
  <li>Cats</li>
  <li>Kittens</li>
</ol>
```
<ol>
  <li>Cats</li>
  <li>Kittens</li>
</ol>

### 3. `<section>` -- Section

* *type:* block-level
* *content:* Flow content
* *support:* all
* *example:*
```
<section>
  <h1>CATS</h1>
  <p>Cats are so cool.<p>
</section>
```
<section>
  <h1>CATS</h1>
  <p>Cats are so cool.<p>
</section>
